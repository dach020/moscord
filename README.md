### Client Directory

### Server Directory

### Setup
To run this project, install it to `client` and `server` directory using npm:

```
$ npm install
$ npm start
```

You can then navigate to:
* `http://localhost:4001/` for the news website.
* `http://localhost:3000/` for the REST APIs
